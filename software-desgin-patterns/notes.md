# Software desgin patterns



## Program:
program is any set of instructions written in code.
It can be small, technical, or invisible to the user.
Might run in the background or be part of something bigger.

## Application 

An application is a type of program designed for end users.
Usually has a user interface (GUI, web UI, mobile UI).
Built to solve a specific user problem.
